/===============================================\
|Sokoball of Osaka Level Editor by Kurtis (2022)|
\===============================================/

This program was written in C# 2008 Express.
The tool requires .Net Framework 3.5 to run

Usage
=====
The tool is not foolproof. If you are not careful
unexpected behaviour may occour in the game.

File->Load - Load a SOK level file
File->Save - Save a SOK level file

The same as the game's built in editor, but with more working elements.
Arrows to move the cursor, the corresponding key to place a level element.
